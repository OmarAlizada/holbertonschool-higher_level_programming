# Python - Hello, World

## Description
This project is the starting point of my Python programming journey at **Holberton School**. It covers the basic syntax, variable assignments, and various ways to manipulate and print strings.

## Learning Objectives
At the end of this project, I learned:
* How to use the Python interpreter.
* How to print text and variables using `print`.
* How to use strings: concatenation, indexing, and slicing.
* How to use f-strings for clean output formatting.
* The importance of following the **Pycodestyle (PEP 8)** for clean and readable code.

## Technologies
* Scripts written in **Python 3.x**
* Tested on **Ubuntu 20.04 LTS**
* Linter: **pycodestyle 2.8.**

## Files
| File | Description |
| --- | --- |
| `0-run` | A shell script that runs a Python script. |
| `3-print_number.py` | Printing an integer with a string. |
| `4-print_float.py` | Printing a float with specific precision. |
| `5-print_string.py` | Practicing string repetition and slicing. |
| `6-concat.py` | Concatenating multiple strings. |
| `7-edges.py` | Extracting parts of a string using indexing. |
